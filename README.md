## 👨‍💻 Sobre mim

Engenheiro de Software formado pela **UTFPR**, com uma trajetória que vai da bancada de manutenção ao quadro branco da sala de aula — e agora, ao código.

Passei pelo suporte técnico, atendimento a sistemas de gestão, manutenção de laboratórios de TI e hoje atuo como **Professor universitário**, ministrando Sistemas Operacionais e Segurança da Informação. Cada experiência me ensinou algo que vai além da técnica: **entender o problema antes de propor a solução.**

Essa bagagem me dá uma visão completa do ciclo de desenvolvimento — do hardware ao software, do usuário final à documentação técnica — o que me torna um profissional mais completo mesmo no início da carreira como desenvolvedor.

## 🚀 Projetos em Destaque

### 🗺️ TurismoApp-API
> Sistema de gerenciamento de Pontos Turísticos com operações completas de cadastro, consulta, edição e remoção.

- **Backend:** C#, ASP.NET Core, Entity Framework Core, SQL Server
- **Frontend:** React, Vite, Tailwind CSS
- **Arquitetura:** Camadas — `Controller → Service → Repository → Database`
